# Homework Exercise 1 - Review The Basics Together

1. Create a fork of this repository
2. Invite your teammate as a collaborator
3. Each teammate should create a branch based of `main` called `feature/{username}`
4. Create a directory on your user name.
5. In that directory create a file called my_feature.
6. Push your changes to the remote repository and create a pull request
7. Ask your teammate to review your PR
8. Merge to main


## Learning Outcomes

- Navigate the CLI with basic command utlities e.g. `ls`,`mv`,`cat`
- Use a built in CLI text editor
- Go through the basic Git pipeline: fork → clone → branch → commit → push → PR → merge
- Navigate GitHub and assign collabrators to review your Pull Requests
