# Homework_2 - Handling Merge Conflicts

1. Create a branch called 'homework/{user-name}-merge-conflict-hw'

2. Teammate A must edit README.md by adding a new line as follows:

```
## Test
This is a test sentence.
```
3. Teammate A must then commit their changes, push, create and create a PR

4. Teammate B must make a change to the same file, push and create a PR

5. Teammate A must then merge their PR

6. This will create a merge conflict in Teammate B's PR, it must be resolved and committed


## Learning Outcomes

- Resolve Merge conflicts between local and remote repositories 
- Understand the difference between incoming and current changes
